The Lousson Project's PEAR Channel

Registering the channel: 
pear channel-discover pear.lousson.org
Listing available packages: 
pear remote-list -c lousson
Installing a package: 
pear install lousson/package_name
Installing a specific version/stability: 
pear install lousson/package_name-1.0.0
pear install lousson/package_name-beta
Receiving updates via a feed: 
http://pear.lousson.org/feed.xml

Lousson_Exception - Standard exception interface and error classes

The Lousson_Exception package consists of a standard exception interface, specific for the Lousson-project, and a bunch of error classes extending PHP and SPL exceptions and implementing the interface.

Install commandpear install lousson/Lousson_Exception

Dependencies pear.phix-project.org/Autoloader, pear.lousson.org/Lousson_Sniffs
MaintainersMathias J. Hennig (as lead)
Releases0.2.0 (stable)

Lousson_Sniffs - Coding-styles and PHP_CodeSniffer integration

The Lousson_Sniffs package is used for both, maintaining the Lousson Project's coding-styles as well as providing an integration with PHP_CodeSniffer.

Install commandpear install lousson/Lousson_Sniffs
